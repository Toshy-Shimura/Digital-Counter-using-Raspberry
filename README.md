# Digital counter using raspberry pico
* Rasberry pi pico W
* Ultrasonic Sensor HC-SR04
* LCD Display 1602  (16x2)

##Requirements
                    
> [MicroPython](https://micropython.org/)


#Ampy

`$ pip install ampy`
`$ ampy --port "COM PORT" put "file"`
